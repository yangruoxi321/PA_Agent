【文件定位】本文件职责：在已确认极速下跌中，给出顺势入场与反转尝试的具体方案。仅在诊断结果为极速下跌时加载。

【核心知识】

一、尖峰期间的基本做空交易策略

策略1：市价单做空入场（Market Order Entry）
- 使用时机：下跌尖峰正在进行中，且没有明显放缓迹象
- 入场点：当前市场价格
- 优点：能够确保入场，不错过行情
- 缺点：可能在低点入场，滑点风险大（恐慌性抛售时尤其严重）
- 执行：设置市价空单后立即确认成交价格
- 注意：恐慌性下跌时滑点可能很大，考虑使用限价单

策略2：止损单做空入场（Stop Order Entry）
- 使用时机：尖峰有加速迹象，但未完全启动
- 入场点：设置在近期低点下方一小距离（如最近趋势棒高度的一半）
- 优点：能够在跌破确认后入场，降低假跌破风险
- 缺点：可能错过最佳入场点
- 执行：设置止损空单，价格跌到触发价后自动入场

策略3：限价单做空入场（Limit Order Entry）
- 使用时机：尖峰已经出现暂停或反弹，希望在更好的价格做空
- 入场点：设置在前一棒的高点上方1-2个跳动点
- 优点：能够在更优价格入场，降低成本
- 缺点：可能无法入场，错过行情
- 执行：在价格反弹到限价位时自动成交
- 提醒：限价单可能完全无法成交，需准备好替代方案

策略4：反弹入场做空（Rally Entry / Pullback Short）
- 使用时机：尖峰结束并出现正常幅度的反弹
- 入场点：反弹的50%-61.8%斐波那契回撤位
- 优点：更好的风险收益比
- 缺点：尖峰可能已经结束，下跌趋势无法恢复
- 需要结合反弹期的表现来判断

二、尖峰到反弹/通道转换时的交易策略

尖峰后的三种发展路径：
1. 趋势恢复为下跌通道（概率最高，约60%）
2. 进入交易区间（概率中等，约30%）
3. 趋势反转（概率最低，约10%，但下跌中反转概率略高于上涨）

交易策略选择：

路径1：下跌通道恢复（高概率）
- 识别特征：反弹幅度<50%，通道倾斜向下，反弹呈波浪形
- 策略：在通道上轨（下降趋势线）附近做空
- 止损位：顺势信号棒/反弹高点上方1跳
- 目标位：通道下轨或测量通道高度候选，最终≤1.5R

路径2：进入交易区间（中等概率）
- 识别特征：反弹幅度>61.8%，出现盘整状态
- 策略：在区间内进行高抛低吸，跌破区间边界时顺势做空
- 止损位：区间信号棒极点外1跳，边界只作结构参考
- 目标位：min(区间宽度50%~80%, 1.5R)

路径3：趋势反转（低概率）
- 识别特征：出现卖出高潮，后续出现明显反转K线
- 策略：等待反转确认后考虑做多
- 止损位：反转信号棒/二次入场低点下方1跳
- 目标位：前期下跌趋势起点候选，最终≤1.5R

三、追空（Chasing the Bear Spike）的条件和风险控制

追空的风险：
- 若使用结构测量目标，止损可能需要比普通信号棒止损更宽；尖峰起点只作为候选放大位，不是默认止损
- 恐慌性下跌中的滑点可能非常大
- 尖峰可能在任何时刻结束（尤其是V形反转）
- 卖出高潮后追空可能是最危险的交易

可追空的条件（全部满足才可追）：
1. 突破非常强势：连续3+根大型空头趋势棒，无明显反弹
2. 小头寸交易：正常头寸的50%或更小
3. 止损规则：默认最近反弹/趋势棒极点外1跳；若结构目标超过1.5R，只能按 widen-stop 放宽，且放大后止损不得超过尖峰高度50%
4. 市价或限价单入场都可，但必须立即入场
5. 市场背景配合：跌破重要技术位/区间边界
6. 单周期多尺度确认：长程结构窗口也看跌
7. 未出现卖出高潮信号

不可追空的情况：
1. 已经出现卖出高潮信号（大型空头趋势棒后的小棒/十字星/长下尾线棒）
2. 尖峰已经在通道内运行了一段时间
3. 尖峰跌破的是交易区间内部的次级低点
4. 交易区间非常紧凑（止损会很小但无效）
5. 没有任何突发消息或技术位跌破支撑
6. 恐慌性抛售已经持续多根棒线，接近高潮

追空的头寸管理：
- 正常头寸：以 1R 为参考规模（具体手数由交易者自定）
- 追空头寸：0.5R（约为正常规模的一半）
- 恐慌性抛售追空：0.25R风险（25%正常头寸）
- 如果追空后行情不利，务必严格止损

四、卖出高潮后的交易策略

卖出高潮的定义：
- 连续2-5根大型空头趋势棒
- 紧接着出现带长下尾线的小型棒线或十字星
- 或出现多头趋势棒确认止跌

卖出高潮后禁止的行为：
- 不在卖出高潮后立即做空（追空高潮）
- 不在卖出高潮后立即设置新的空头止损单
- 不在卖出高潮后增加空头头寸

卖出高潮后的正确策略：
1. 持有空仓的处理：如果已有空仓，至少减仓50%
2. 获利的处理：在高潮信号出现后开始部分获利了结
3. 等待的处理：等待至少10根棒线，等待反弹结束
4. 反转交易：等待反转信号确认后再考虑做多
5. 观察策略：不做任何交易，等待新的结构形成

卖出高潮后的最佳做空入场点：
- 第一个反弹的低点被跌破后考虑做空
- 或者等待二次反弹失败后做空
- 或者等待价格形成新的交易区间后进行区间交易
- 注意：反弹后如果创新低，可能形成二次尖峰

五、下跌趋势日和开盘起下跌趋势日的交易

开盘起下跌趋势日的定义：
- 价格在开盘后的前几根棒线内形成当天的最高点
- 开盘区间小于当天区间的25%
- 开盘区间被向下突破后，价格朝突破方向持续运动
- 与上涨趋势日类似，约20%的交易日属于开盘起下跌趋势日

开盘起下跌趋势日的交易步骤：
1. 识别开盘区间：开盘后前3-6根K线形成的价格范围
2. 等待跌破：价格跌破开盘区间下边界
3. 确认信号：跌破后至少有一根空头趋势棒确认
4. 入场：在跌破后的反弹中入场做空
5. 止损：放在开盘区间上侧边界外

下跌趋势日的特征：
- 开盘后迅速形成当天的最高点
- 反弹幅度极小（<30%日均波幅）
- 每次反弹都是好的做空入场点
- 价格可能收在接近最低点的位置

下跌趋势日的交易规则：
- 如果开盘低点在第一小时被突破，趋势日失败
- 如果开盘高点在第一小时被突破，趋势日失败
- 趋势日确立后，反弹是好的做空入场点
- 下跌趋势日通常在欧美交易时段交界处最易发生

六、小幅反弹趋势日（Trend Day with Small Pullback）的做空交易

小幅反弹趋势日的特征：
- 最强的下跌趋势日类型，一个月可能只出现1-2次
- 当天的反弹幅度仅为日均波幅的10%-30%
- 所有的反弹都是好的做空入场点
- 空头趋势棒和反弹棒交替出现，呈"台阶式"下跌

小幅反弹趋势日的识别：
- 开盘后迅速形成当天的最高点
- 价格快速下跌后出现小幅反弹
- 反弹棒明显小于趋势棒（约30%-50%）
- 反弹时间很短（1-3根棒线）
- 恢复下跌后继续创新低

小幅反弹趋势日的做空策略：
1. 首次反弹入场：在第一次明显反弹时做空
2. 每次反弹都是机会：价格每次反弹到下降趋势线都是做空机会
3. 止损设置：放在最近反弹高点上方
4. 目标：持续持有直到出现反转信号
5. 加仓：在下跌趋势恢复后可以在反弹点加仓

七、止损设置原则

止损类型1：尖峰结构止损 / widen-stop 候选
- 默认定位：最近反弹K线或最近趋势棒高点上方1跳
- 尖峰起点：只作为 widen-stop 候选放大位，用于让结构目标满足≤1.5R
- 硬上限：放大后止损不得超过尖峰高度50%；超过则放弃
- 适用：3根以上标准尖峰且结构目标明确时

止损类型2：信号棒过长时的紧缩止损
- 定位：当标准图表止损过远时，在信号棒极点外设较紧止损
- 优点：限制单笔价格风险距离
- 缺点：可能设在非关键结构位，易被正常波动触发
- 适用：信号棒明显过长、仍想评估入场时

止损类型3：技术位止损
- 定位：将止损放在关键技术位上方/下方
- 技术位：前高、下降趋势线、均线、斐波那契位
- 优点：反映市场结构变化
- 缺点：需要准确识别技术位
- 适用：有一定技术分析基础时

止损设置的安全规则：
- 永远不要将止损放在精确的最高价/最低价
- 至少留出1-2个跳动点的缓冲空间
- 如果止损距离超过50个跳动点，考虑缩小头寸
- 止损一旦设置，不要因为害怕被扫而移动

做空止损的特殊注意事项：
- 恐慌性下跌中，价格可能在止损位附近剧烈波动
- 考虑使用"限价止损"（Stop Limit Order）避免大滑点
- 或者使用更宽的止损 + 更小的头寸来应对波动

八、利润目标设置

目标类型1：测量运动目标（Measured Move）
- 计算方法：从尖峰起点测量到尖峰终点
- 第一目标：测量运动的100%（等距）候选
- 第二目标：测量运动的138.2%或161.8%候选
- 约束：任何候选目标若导致回报÷风险>1.5，必须用尖峰 widen-stop 适配；无法适配则放弃，不输出>1.5R目标
- 优点：有客观的计算依据
- 缺点：可能过于机械

目标类型2：通道高度目标
- 计算方法：测量通道的高度作为目标
- 通道高度 = 通道上轨 - 通道下轨（绝对值）
- 目标位 = 通道下轨 - 通道高度（或倍数）
- 适用：尖峰后进入通道的情况

目标类型3：技术位目标
- 识别前一个重要的支撑位/阻力位
- 将其作为利润目标
- 常见技术位：前低、前高、整数心理位、均线

目标类型4：分批出场
- 第一批：在达到第一个结构目标（1R~1.5R）时出场50%
- 第二批：在达到第二个结构目标（通道边界/测量移动，仍不超过1.5R）时出场25%
- 第三批：剩余25%用跟踪止损（移至盈亏平衡后跟随波段极点）
- 优点：锁定部分利润的同时保留趋势参与

⚠️ 测量运动目标 > 1.5R 时的适配规则（widen-stop）：
- 尖峰的测量运动目标（100%/138.2%/161.8%投影）通常远超1.5R。
- 禁止缩小止盈目标来迎合1.5R——结构目标具有磁力效应，过早止盈降低系统优势。
- 通过放大止损来适配：使结构目标 ≤ 1.5 × 止损距离。
  例：结构目标30点 → 止损≥20点（此时30/20=1.5R）。
- 若放大后的止损 ≤ 尖峰高度50% 且通过交易者方程 → 允许下单。
- 若放大后的止损 > 尖峰高度50% → 放弃，不强行入场。
- 此规则与「二元决策.txt §10.3」尖峰适配规则统一；本文不再使用"比正常宽50%-100%"作为独立阈值。
- 详见「二元决策.txt §10.3」"尖峰交易与1.5R上限的适配规则（widen-stop）"。

九、逐步出场策略

出场原则：分批出场优于一次性出场

策略1：固定比例出场
- 在价格达到入场价的1R时，买入平仓50%空头头寸
- 剩余头寸移动止损到盈亏平衡点
- 继续持有直到出现反转信号
- ⚠️ 尖峰场景中第一批1R~1.5R为结构目标；若结构目标 > 1.5R，
  则用 widen-stop 放大止损使结构目标 ≤ 1.5R，详见上文八、目标类型4。
- 剩余头寸移动止损到盈亏平衡点
- 继续持有直到出现反转信号

策略2：技术位移动止损
- 随着价格朝有利方向移动，将止损移动到：
- 第一个盈利目标位：移动到入场价
- 第二个盈利目标位：移动到第一个目标位
- 以此类推，保护已有利润

策略3：趋势棒减仓法
- 当出现连续3根空头趋势棒时，减仓50%
- 原因：趋势可能即将结束
- 剩余头寸继续持有到反转信号

策略4：反转信号出场
- 当出现卖出高潮信号时，平仓50%-75%
- 当出现反转K线组合时，平仓全部
- 等待新的交易机会

十、连续卖出高潮后的交易策略

连续卖出高潮的定义：
- 第一个尖峰后进入反弹
- 反弹期后再次出现尖峰（第二个高潮）
- 两次高潮间隔至少10棒

连续卖出高潮后的交易规则：
- 在第二个高潮后不要追空
- 考虑在第二个高潮后减仓或了结
- 等待反弹后再考虑新的空头入场
- 如果出现第三个高潮，坚决出场或反手做多

连续卖出高潮后的最佳策略：
1. 第二个高潮后开始逐步获利了结
2. 等待至少10根棒线的反弹
3. 在反弹的高点重新做空
4. 或者等待新的尖峰信号再次做空

第三个高潮的特别处理：
- 第三次高潮是最可能的底部反转点
- 如果持有空头仓位，在第三个高潮出现时全部平仓
- 考虑在第三个高潮后建立多头仓位
- 第三个高潮后的反转往往是最确定的反转信号

十一、下跌尖峰中做多的逆势策略（仅限老手）

警告：逆势做多风险极高，新手禁止尝试！

逆势交易的前提条件（必须全部满足）：
1. 你是经验丰富的交易者（至少3年以上实盘经验）
2. 已经出现明确的卖出高潮信号
3. 单周期多尺度都显示价格处于低位
4. 有明确的止损位和利润目标
5. 头寸仅为正常头寸的25%-50%

逆势做多的入场点：
- 卖出高潮的第一根反转棒的低点被突破后
- 或者在第二个回调低点失败后（不再创新低）
- 或者在价格形成新的交易区间下轨买入
- V形反转的第一个回调是最佳入场点

逆势做多的止损：
- 必须放在近期低点下方
- 绝不能使用"紧密止损"——底部反转往往需要空间
- 最终仍必须通过1.5R上限；若所需结构止损过远导致目标无法≤1.5R，则放弃

逆势做多的目标：
- 第一个目标：1R或最近结构阻力
- 第二候选：尖峰起点、前一个上涨趋势高点、长程结构窗口阻力位
- 所有候选最终不得超过1.5R

逆势做多的注意事项：
- 卖出高潮后的V形反转是最可靠的逆势做多信号
- 但如果反转失败（价格再次创新低），必须立即止损
- 不要试图"抄底"——等待反转信号确认后再入场
- 逆势做多的头寸管理必须比顺势交易更加保守

【实战规则】

规则1：入场时机规则
- 只在尖峰进行中或反弹到位的节点入场做空
- 不在尖峰启动前猜测入场
- 不在尖峰结束后追空入场（除非是反弹入场）

规则2：头寸管理规则
- 追空时建议轻仓（约正常规模一半）
- 恐慌性抛售追空时头寸减至25%
- 逆势做多头寸减至25%-50%

规则3：止损执行规则
- 止损一旦触发，无条件执行
- 绝不允许手动移动止损"再等等"
- 如果频繁被扫，考虑减少头寸而不是放宽止损

规则4：出场纪律规则
- 没有达到盈利目标前，不要因为"害怕失去"而提前出场
- 出现反转信号时，务必减仓或了结
- 分批出场时，不要随意调整已设定的出场比例

规则5：禁止交易规则
- 卖出高潮后不做空
- 假跌破后不做空
- 没有明确止损位的情况下不做空
- 单周期多尺度不一致时不做空
- 恐慌性抛售接近尾声时不追空

规则6：交易日志规则
- 记录每一笔交易的入场理由
- 记录入场时的价格和设置的止损
- 记录出场时的价格和理由
- 定期复盘，总结经验教训

【例外与陷阱】

陷阱1：假跌破后入场做空
- 跌破交易区间后很快反弹
- 尖峰"失败"，进入交易区间或反转
- 解决：等待至少2根棒线确认跌破有效

陷阱2：追空过深
- 在尖峰已经运行3-5棒后才追入
- 止损距离过远，风险收益比不佳
- 解决：坚持追空条件，只追强势尖峰

陷阱3：止损放在错误位置
- 止损放在尖峰起点但尖峰刚启动
- 或者止损放在太近的位置被假跌破扫掉
- 解决：根据尖峰的发展阶段选择合适的止损类型

陷阱4：提前获利了结
- 在获得1R利润后立即全部平仓
- 错失后面更大的下跌趋势
- 解决：使用分批出场策略

陷阱5：逆势做多的过度自信
- 认为自己能"预测"底部
- 逆势加仓，摊平成本
- 结果：大幅亏损
- 解决：绝对禁止逆势加仓，逆势交易头寸限制在25%

陷阱6：忽视单周期多尺度信号
- 只看即时信号窗口，忽视长程结构窗口的趋势
- 在长程结构窗口支撑位附近做空
- 解决：入场前检查长程结构窗口

陷阱7：恐慌性抛售中的冲动交易
- 看到暴跌就冲动做空
- 没有交易计划，没有止损
- 结果：在底部做空被V形反转套牢
- 解决：恐慌性抛售中应冷静观望，等待反弹后再做决策

陷阱8：卖出高潮信号识别错误
- 将普通的反弹误认为卖出高潮
- 过早平仓空单，错失后续下跌
- 解决：严格按照卖出高潮的5项标准来判断
