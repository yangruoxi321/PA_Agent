【文件定位】本文件职责：判断通道宽度（窄/宽），给出对应交易策略调整。仅在诊断结果为通道态且需要宽度判断时加载。

【核心知识】

一、通道的基本定义与分类

通道（Channel）是指价格在两条大致平行的趋势线之间运行的形态。根据回撤深度和结构稳定性，Al Brooks体系在本项目中统一分为三大类。**全系统以「市场诊断框架.txt」统一判定树为准：最近一波回撤百分比是主分类键，斜率和视觉宽度只作辅助描述。**

1. 窄/紧凑通道（Tight Channel）
   - 定义：最近一波回撤 <30%，回撤短且浅，K线实体连贯、几乎无重叠的通道
   - 辅助描述：斜率大（常大于45度）、线距窄、1~3根/跳动级微回撤
   - 特征：连续的趋势棒（Trend Bar）组成，每根棒的回撤不超过前棒的1/3，收盘价持续创新高/新低
   - 动量特征：极强的方向性动量，市场参与者几乎一边倒
   - 时间特征：通常持续5-20根K线，极少数超过30根
   - 本质：在长程结构窗口上，窄通道往往对应一个尖峰（Spike）阶段

2. 常规通道（Normal Channel）
   - 定义：最近一波回撤 30%~50%，波段清晰，顺势仍占优但已不是尖峰级强趋势
   - 交易含义：顺势为主，只有在通道边界出现明确反转信号时才允许小仓逆势刮头皮

3. 宽通道（Broad Channel）
   - 定义：最近一波回撤 50%~78.6%，波段大、有明显的回撤和反弹、K线之间有较多重叠的通道
   - 辅助描述：斜率小（常小于30度）、趋势线和通道线之间距离较宽，价格在其中做锯齿形运动
   - 特征：趋势线和通道线之间距离较宽，价格在其中做锯齿形运动
   - 别称：台阶（Stairs）——因为价格像走台阶一样一步步前进
   - 动量特征：多空双方都有力量，但一方略占优势
   - 时间特征：可持续数十根至上百根K线
   - 本质：宽通道是倾斜的交易区间（Inclining/Declining Trading Range），与交易区间共享同一套策略框架（顺势方向70%仓位，边界反转+突破测试），区别仅在于有可识别的斜率和可画出的通道线

   ⚠️ 宽通道 vs trending_tr 判定规则（统一参照「市场诊断框架.txt」）：
   - 有3组以上 HH+HL / LL+LH，且可画出大致平行的通道线 → broad_channel
   - 有方向偏好但序列不足3组，或通道线不稳定 → trending_tr
   - 两者策略框架相近；边界模糊时按统一判定树裁决：3组以上且可画稳定通道线 → broad_channel；只有2组或通道线不稳定 → trending_tr

4. 微型通道（Micro Channel）
   - 定义：由2-10根K线组成的极短、极紧凑通道，几乎没有回撤，属于窄通道的最极端形式
   - 特征：多数K线的高点和低点沿同方向推进，几乎无回撤；允许1到2个很小的内包/十字暂停，但不得破坏整体单向推进
   - 动量特征：极端的方向性动量，通常是更大趋势中的一段加速
   - 本质：微型通道是窄通道的极端形式，突破后通常失败

5. 台阶形态（Stairs）
   - 定义：宽通道中价格呈台阶式推进的形态，每上一个台阶后回撤，再上第二个台阶
   - 分为：上升台阶（Bull Stairs）和下降台阶（Bear Stairs）
   - 特征：几乎每次突破后都有突破测试（Breakout Test）

6. 收缩台阶（Contracting Stairs）
   - 定义：每个后续突破的幅度小于前一个突破的幅度
   - 特征：台阶逐渐变小，表明动能衰退
   - 信号意义：趋势可能即将结束，准备反转

7. 加速台阶（Accelerating Stairs）
   - 定义：后续突破幅度大于前一个突破
   - 特征：台阶逐渐变大，动能加速
   - 信号意义：可能突然加速突破趋势通道，形成买入/卖出高潮

二、窄通道与宽通道的详细对比

| 特征 | 窄/紧凑通道（Tight） | 常规通道（Normal） | 宽通道/台阶（Broad/Stairs） |
|------|------------------------|---------------------------|---------------------------|
| 主判定：最近回撤 | <30% | 30%~50% | 50%~78.6% |
| 斜率 | 辅助：大（常>45°） | 辅助：适中 | 辅助：小（常<30°） |
| K线重叠 | 几乎无 | 中等 | 较多 |
| 趋势棒比例 | >65% | 约50%上下 | 30-50% |
| 交易方向 | 只做顺势 | 顺势为主，少量逆势 | 可双向交易 |
| 突破特征 | 第一个逆势突破通常失败 | 等回撤/测试 | 几乎每次突破后都有测试 |
| 长程结构窗口对应 | 尖峰（Spike） | 通道期 | 倾斜交易区间 |
| 反转概率 | 低，先等趋势线突破 | 中等 | 中，顺势仓位应占70% |

硬规则：斜率角度、绝对跳动数、通道视觉宽窄不得覆盖回撤百分比主分类。

三、通道转换信号

1. 窄通道→宽通道（趋势减速）
   - 信号：出现较大的反向K线、回撤开始触及趋势线、K线开始重叠增多
   - 含义：趋势从尖峰阶段进入通道阶段，动量减弱但仍维持方向
   - 操作调整：从只做顺势转为可做双向，开始关注逆势反转信号

2. 宽通道→窄通道（趋势加速）
   - 信号：回撤越来越小、K线实体变大、开始出现连续趋势棒
   - 含义：趋势加速，可能形成新的尖峰
   - 操作调整：停止逆势交易，只做顺势

3. 宽通道→交易区间
   - 信号：价格在通道上下边界之间来回震荡，无明显趋势方向
   - 含义：趋势结束，进入区间震荡
   - 操作调整：低买高卖，不追突破

4. 任何通道→反转
   - 信号：通道被强势突破、出现强反转棒、形成二次入场反转架构
   - 含义：趋势方向改变
   - 操作调整：准备逆势交易

四、微型通道的特殊性质

1. 微型通道的形成
   - 在强趋势中出现，通常是窄通道中最紧凑的部分
   - 可能出现在趋势开始（突破后的加速）、趋势中段（动能爆发）或趋势末端（高潮）

2. 微型通道的突破
   - 突破通常失败！这是微型通道最重要的交易特征
   - 突破失败后，价格通常会回到通道内并沿原方向继续运行
   - 因此，突破失败是顺势入场的好机会

3. 微型通道的交易窗口
   - 在微型通道形成过程中，不要试图逆势入场
   - 等待突破发生，如果突破失败，顺势入场
   - 如果突破成功且伴随强动量，可能是趋势反转的信号

五、台阶形态的深入分析

1. 上升台阶（Bull Stairs）
   - 价格创新高→回撤至前期高点附近→再次创新高
   - 每个回撤都是买入机会
   - 回撤通常在前期波段高点附近获得支撑

2. 下降台阶（Bear Stairs）
   - 价格创新低→反弹至前期低点附近→再次创新低
   - 每个反弹都是卖出机会
   - 反弹通常在前期波段低点附近遇到阻力

3. 台阶与突破测试
   - 宽通道/台阶中，几乎每次突破后都会回来测试突破点
   - 这是宽通道最重要的交易特征
   - 突破测试成功（价格回到突破点后继续原方向）= 顺势入场机会
   - 突破测试失败（价格回到突破点后反向突破）= 反转信号

【分析流程】

步骤1：确定长程结构窗口趋势方向
- 查看K100-K1长程结构窗口
- 判断长程结构窗口是上涨趋势、下跌趋势还是交易区间
- 近期结构方向决定了顺势交易的主方向，长程结构窗口方向作为风险参考

步骤2：识别当前通道类型
- 观察K线排列：连续趋势棒→窄通道；交错阴阳棒→宽通道
- 测量最近一波回撤占前一波推进的比例：<30% tight，30%~50% normal，50%~78.6% broad
- 斜率、触线次数、绝对跳动数只作辅助描述
- 判断结论：微型通道、窄/紧凑通道、常规通道、宽通道/台阶

步骤3：评估通道所处阶段
- 是通道的初期、中期还是末期？
- 是否出现通道转换信号？
- 台阶形态是否在收缩（动能衰退）还是加速？

步骤4：制定交易策略
- 窄通道：只做顺势，等待回撤入场或突破失败入场
- 宽通道/台阶：可做双向，但顺势交易优先
- 微型通道：等待突破失败后顺势入场
- 收缩台阶：准备趋势反转，注意二次入场机会

步骤5：确定入场点、止损和目标
- 根据通道类型和阶段选择入场时机
- 设置保护性止损：默认信号棒极点外1跳；宽通道噪声大时可扩至最近波段极点外1跳
- 确定目标位：通道宽度、对边或测量运动只作候选；最终目标必须取不超过1.5R的最近结构位

步骤6：跟踪和调整
- 监控通道是否出现转换信号
- 根据走势调整止损和目标
- 通道转换时及时调整交易策略

【实战规则】

一、窄通道交易规则

1. 入场规则
   - 顺势入场：等待回撤至趋势线附近入场（但窄通道回撤极少，可能等不到）
   - 突破失败入场：当价格突破趋势线后迅速回到通道内，这是最佳入场点
   - 微型通道突破失败：在2-10棒微型通道中，突破失败是高概率顺势入场机会
   - 禁止逆势入场：窄通道中绝不做逆势交易

2. 止损规则
   - 顺势入场：止损设在信号棒极点外1跳动
   - 突破失败入场：止损设在突破极点外1跳动
   - 通道极点止损：如果通道运行较长，可将止损移至最近波段极点外1跳动

3. 止盈规则
   - 窄通道通常持续5-20棒，可在通道出现减速信号时减仓
   - 目标位：通道宽度（起点到通道线的距离）的等幅测量候选；最终不得超过1.5R
   - 如果窄通道演变为宽通道，继续持有但调整策略

4. 特殊注意
   - 窄通道的第一个突破通常失败——不要在第一次突破时逆势入场
   - 但如果突破伴随极强动量（大实体、多根趋势棒），可能是趋势反转的信号
   - 窄通道在长程结构窗口上通常是尖峰，意味着更强的趋势方向

二、宽通道/台阶交易规则

1. 入场规则
   - 顺势入场：台阶回撤至前期极点附近入场
   - 逆势入场：在通道边界附近出现反转信号时，可做逆势交易（但需谨慎）
   - 突破测试入场：突破后回测突破点，测试成功则顺势入场
   - 双向交易：宽通道允许在通道上界做空、下界做多

2. 止损规则
   - 顺势入场：止损设在回撤极点外1跳动
   - 逆势入场：止损设在信号棒极点外1跳；通道边界只作结构参考，不能替代信号棒止损
   - 突破测试入场：止损设在测试低/高点外1跳动

3. 止盈规则
   - 顺势目标：下一个台阶目标或通道对边候选，最终≤1.5R
   - 逆势目标：通道中位数或对边候选，最终≤1.5R
   - 突破测试成功：目标为通道宽度或下一个台阶候选，最终≤1.5R

4. 台阶形态特殊规则
   - 上升台阶中每个回撤都是买入机会
   - 下降台阶中每个反弹都是卖出机会
   - 台阶幅度相等时趋势最健康
   - 台阶收缩时准备反转
   - 台阶加速时注意可能的突破

三、微型通道交易规则

1. 入场规则
   - 在微型通道形成过程中：不入场
   - 突破失败后：顺势入场（高概率交易）
   - 突破成功+强动量：可能预示反转，等二次确认

2. 止损规则
   - 突破失败入场：止损设在突破极点外1跳动
   - 如果信号棒过长，优先等待更小信号或放弃；资金管理止损（8跳动或信号棒60%高度）只作为风险上限参考，不能替代结构失效位

3. 止盈规则
   - 目标位：通道宽度的等幅测量候选，最终≤1.5R
   - 或使用跟踪止损，创新高/低后移至最近波段极点

四、收缩台阶交易规则

1. 识别收缩台阶
   - 每个后续突破的幅度小于前一个
   - 动能指标可能背离
   - 回撤可能开始变大

2. 交易策略
   - 减少顺势仓位，因为趋势动能衰退
   - 关注反转信号，准备逆势入场
   - 收缩台阶中最后一个台阶可能是陷阱——看似趋势延续但动能不足

五、通道转换的交易规则

1. 窄通道→宽通道
   - 停止只做顺势的限制
   - 开始关注双向机会
   - 利用突破测试进行交易

2. 宽通道→窄通道
   - 停止逆势交易
   - 只做顺势，利用回撤入场
   - 注意可能的趋势加速

【例外与陷阱】

1. 窄通道陷阱
   - 假突破陷阱：价格突破趋势线后迅速回到通道内，诱骗逆势交易者入场
   - 趋势持续陷阱：窄通道持续时间超出预期，逆势交易者反复被打止损
   - 末端加速陷阱：窄通道末端突然加速，看起来像要突破实际上是买入/卖出高潮

2. 宽通道陷阱
   - 假突破陷阱：宽通道中的突破经常失败并回到通道内
   - 双向交易陷阱：在宽通道中做双向交易时，容易被两边打止损
   - 区间扩张陷阱：宽通道可能逐渐扩张，导致通道边界不断被更新

3. 微型通道陷阱
   - 反转陷阱：微型通道的突破有时确实成功，不是所有突破都会失败
   - 时间陷阱：微型通道可能持续时间很短，错过入场窗口
   - 高潮陷阱：微型通道出现在趋势末端时可能是高潮，突破后趋势反转

4. 台阶形态陷阱
   - 等幅台阶陷阱：不是所有台阶都等幅，有时台阶会突然变大或变小
   - 收缩台阶陷阱：收缩台阶可能只是暂时休整，之后继续趋势
   - 加速陷阱：台阶加速可能是买入/卖出高潮的前兆

5. 通道转换陷阱
   - 过早判断转换：将正常的回撤误判为通道转换
   - 滞后判断转换：通道已经转换但策略没有及时调整
   - 多级转换：窄通道→宽通道→交易区间→反转，可能需要多次调整

6. 通用陷阱
   - 过度依赖通道线：通道线是主观的，不同画法可能得出不同结论
   - 忽略长程结构窗口：在即时信号窗口看到的通道可能只是长程结构窗口的一根K线
   - 机械套用规则：市场是动态的，通道形态会演化和变异
